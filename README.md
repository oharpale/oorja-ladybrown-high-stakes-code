# **Controls**
L1: Intake On/Off (normal intake)

L2: Intake Reverse/Off

R1: Score with Ladybrown (load, score, and return to rest position)

R2: Ladybrown in Rest Position

A: Extend and Retract Doinker

B: Clamp and Unclamp Mobile Goal

### Notes
Don't need to turn intake off to make it reverse and vice versa.

The bot will automatically recognize rings of the opposite color and flip them over the bot's mobile goal. The driver will not have control of the intake over that time. If the intake is running normally prior to the color sorting mechanism, after the color sorting is finished, the intake will run normally again without driver action. As soon as color sorting is finished, the hijack will end and the driver will be able to control the intake once more.
